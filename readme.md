# Verra Task Tracker

Verra Task Tracker is an online application designed for employees to conveniently record the volume of completed work. Users can create an account to manage their task records, which are stored in a database using Google Firebase. This project provides a user-friendly interface for adding and deleting task records.

## Features

- User authentication: Create an account or log in with existing credentials.
- Secure data storage: Utilizes Google Firebase to store user accounts and task records.
- Real-time data synchronization.
- User-friendly and responsive web interface.
- Record tasks: Add details of completed work, including volume and description.
- Delete tasks: Remove task records that are no longer needed.

## Usage

1. Open the index.html file in a web browser.
2. Sign up for an account or log in using existing credentials.
3. Add task records by specifying the volume of completed work and a brief description.
4. View your task records and delete them as needed.

## Technologies Used

- HTML, CSS, and vanilla JavaScript for the front-end interface.
- Google Firebase for user authentication and data storage.
